Для запуску тестів спочатку переконатися що:

- встановлений maven не нижче версії 3.1.0;
- файли *driver.exe знаходяться відповідно до config.properties файлу;

Для безпосереднього запуску тестів і звітів:
- в консолі Intelij IDEA (або в cmd папки проекту) ввести команду
    a) для першого запуску
        mvn clean install site jetty:run
    b) якщо вже створений артифакт проекту
        mvn clean test site jetty:run
Для перегляду звітів перейти вбраузері по адресі
- localhost:8080
